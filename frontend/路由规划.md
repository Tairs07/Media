# 前端路由规划

## 路由结构

### 公开路由（无需认证）

1. **首页**
   - 路径: `/`
   - 组件: `Home.vue`
   - 说明: 展示所有公开的媒体内容

2. **登录页**
   - 路径: `/login`
   - 组件: `Login.vue`
   - 说明: 用户登录

3. **注册页**
   - 路径: `/register`
   - 组件: `Register.vue`
   - 说明: 用户注册

4. **媒体详情页**
   - 路径: `/media/:id`
   - 组件: `MediaDetail.vue`
   - 说明: 查看单个媒体文件详情

### 需要认证的路由

5. **上传页**
   - 路径: `/upload`
   - 组件: `Upload.vue`
   - 说明: 上传图片和视频
   - 需要认证: 是

6. **个人中心**
   - 路径: `/profile`
   - 组件: `Profile.vue`
   - 说明: 查看和管理个人信息
   - 需要认证: 是

7. **我的上传**
   - 路径: `/profile/my-uploads`
   - 组件: `MyUploads.vue`
   - 说明: 查看自己上传的所有文件
   - 需要认证: 是

8. **编辑资料**
   - 路径: `/profile/edit`
   - 组件: `EditProfile.vue`
   - 说明: 编辑个人信息
   - 需要认证: 是

## 路由守卫

### 认证守卫
- 检查 `localStorage` 中的 token
- 未登录用户访问需要认证的页面时，重定向到 `/login`
- Token 过期时，清除本地存储并跳转到登录页

## 路由元信息

```typescript
{
  path: '/upload',
  name: 'Upload',
  component: Upload,
  meta: {
    requiresAuth: true,
    title: '上传文件'
  }
}
```
